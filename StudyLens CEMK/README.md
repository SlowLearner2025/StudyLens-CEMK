# SnapLearn CEMK

Lightweight visual study site for MAKAUT CSE (1st Sem). Interactive visualizations and quick quizzes for Physics, Mathematics and Basic Electrical engineering.

---

## Quick preview
- Open `index.html` in your browser (double-click or use a local static server).
- Recommended local preview server (from project root):

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

---

## Project structure (important files)
- `index.html` — landing / subject chooser
- `physics.html`, `math.html`, `electrical.html` — subject pages with visualizations
- `script.js` — shared JS utilities, visualization initialization
- `style.css` — site styles
- `assets/` — place images and other media here (see assets/README.txt)
